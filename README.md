GPA-analysis
A Python-based weighted GPA calculator & academic performance analyzer.

Project Overview
This program provides three core features for student grade management:
1. Calculate weighted GPA automatically based on course credits and exam scores
2. Analyze each subject’s performance to find strong & weak courses
3. Simulate GPA improvement after changing individual course grades for study planning
   
How to Use
1. Unzip GPAanalysis_Xinyi.zip to get the source code
2. Prepare your grade spreadsheet (CSV / Excel with course name, credit hours, score)
3. Run the main script and import your grade file
4. Generate final GPA, subject analysis report, and grade-boost simulation result
